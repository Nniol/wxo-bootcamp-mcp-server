01 — MCP Tools
Registered Tool Functions
All 8 tools exposed by the FastMCP server instance
👤
wxoGetPatientInformation
Get Patient Information
Returns basic patient information from the healthcare database including personal details, contact information, and insurance data. Returns None if patient not found.
Input
patient_id string
PAT + 3 digits
Returns
Patient | None
Demographics, insurance
📋
wxoGetMedicalInformation
Get Medical Information
Returns comprehensive patient medical information — diagnoses (conditions), allergies, and active prescriptions. Returns None if patient not found.
Input
patient_id string
Returns
MedicalServerOutput | None
Conditions, allergies, Rx
💓
wxoGetVitalSignsInformation
Get Vital Signs
Returns patient vital signs from the bedside monitoring system. Calculates creatinine clearance using the Cockcroft-Gault formula given patient age and sex.
Input
patient_id string
age int
sex M | F | OTHER
Returns
VitalSigns | None
BP, HR, BMI, creatinine
💊
wxoGetDrugInformation
Get Drug Information
Returns comprehensive pharmaceutical information for a given drug name, including indications, contraindications, dosage, side effects, and pregnancy category.
Input
drug_name string
Case-insensitive lookup
Returns
DrugInformation | None
Indications, dosage, Rx class
⚠️
wxoGetDrugInteractions
Check Drug Interactions
Identifies potential interactions between two drugs. Lookup is symmetric — order of drug names does not matter. Returns severity, mechanism, clinical effect, and management guidance.
Input
drug_name_1 string
drug_name_2 string
Returns
DrugInteraction | None
Severity, mechanism, mgmt
🚫
wxoGetDrugContraindications
Get Drug Contraindications
Returns contraindication rules for a given drug — conditions under which the drug must not be prescribed, including severity level and clinical rationale.
Input
drug_name string
Case-insensitive lookup
Returns
ContraindicationRule | None
Drug, condition, severity, reason
🔄
wxoGetDrugAlternativeTreatments
Get Alternative Treatments
Retrieves alternative treatment options for a given drug. Useful when a contraindication or interaction prevents use of a first-line medication.
Input
drug_name string
Case-insensitive lookup
Returns
AlternativeTreatment | None
Alternative drug options
🌐
wxoGetPatient360
Get Patient 360 View
Combines all data sources into a single comprehensive patient view — demographics, full medical history, and current vital signs (with creatinine clearance calculated inline).
Input
patient_id string
Returns
Patient360
Composite of all records

05 — Usage Patterns
Common Clinical Workflows
Recommended multi-tool call sequences for agent implementations
🎯
Complete Patient Assessment
  1. Retrieve full patient overview
    wxoGetPatient360
  2. Enumerate all active prescriptions from medical record
    wxoGetMedicalInformation
  3. Cross-check all drug pairs for interactions
    wxoGetDrugInteractions
  4. Verify per-drug contraindications against conditions
    wxoGetDrugContraindications
  5. Generate clinical summary with safety flags
🔍
Drug Safety Analysis
  1. Fetch detailed drug information
    wxoGetDrugInformation
  2. Check contraindications for the drug
    wxoGetDrugContraindications
  3. Check interactions with co-prescribed medications
    wxoGetDrugInteractions
  4. If unsafe, retrieve alternatives
    wxoGetDrugAlternativeTreatments
  5. Provide ranked safety recommendations
⚡
Quick Clinical Check
  1. Get current vital signs (incl. creatinine clearance)
    wxoGetVitalSignsInformation
  2. Review active prescriptions
    wxoGetMedicalInformation
  3. Check for immediate interaction risks
    wxoGetDrugInteractions
  4. Flag critical contraindications
    wxoGetDrugContraindications
  5. Surface real-time clinical alerts
